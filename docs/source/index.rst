Welcome to MicroGridsPy's documentation!
===================================

**MicroGridsPy** is a bottom-up, open-source optimization model, running on Pyomo, a Python library used to model optimisation problems, whose primary goal is to offer an open-source approach to the issue of energy scaling and dispatch in micro grids in remote locations.


Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
